# Terraform EKS

## About

The project creates a EKS cluster with core components.

## Core Components

### Addons

- ALB Controller
- Cluster Autoscaler
- CSI
- Keda
- MetaGPU
- Nginx Controller
- Secrets Store: CSI Driver and Secrets Manager

## Auth

- Assume Role
- SSO

## Node Groups

- General
- GPU
- Obervability

## Observability

- CloudWatch Metrics
- Grafana
- NVidia Exporter
- AWS OpenSearch
- Prometheus
- SQL Exporter

## Requirements

- AWS Admin role.
